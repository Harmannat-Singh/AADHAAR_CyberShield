# AADHAAR_CyberShield
A repo for the Frontend U/UX of the AADHAAR Association's CyberShield Project, my undertaking for my Summer Internship 2025

I worked under AADHAAR assocaition, an NGO, to create a website frontend for their CyberShield Project, aimed at improving digital safety and cyber-hygiene.

## Gist of how the project went:
* Learned CSS, basic Javascript, HTML
* Picked up project from a previous intern
* Tried tinkering, Created logo (learned Inkscape and Figma for it). Not much change overall
* Complete overhaul of the website- used templates to create a professional-looking website based on specificaions and feedback received from mentors
* Worked out Pretty good all-in-all
